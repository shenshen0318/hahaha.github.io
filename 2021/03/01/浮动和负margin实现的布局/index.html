浮动和负margin实现的布局

发表于2021-03-01|更新于2021-03-03|CSS
|字数总计:986|阅读时长:4分钟|阅读量:

圣杯布局和双飞翼布局 通俗的来说就是左右两栏固定宽度，中间部分自适应的三栏布局。

一、圣杯布局

圣杯布局的中间部分在DOM结构上优先，以便先行渲染

  • 实现步骤：
  1. 首先把 left、main、right 都放出来
  2. 给它们三个设置上float: left, 脱离文档流；
  3. 一定记得给 wrap 设置上 overflow: hidden; 可以形成BFC撑开文档
  4. left、right 设置上各自的宽度
  5. main 设置width: 100%;

接下来比较重要了：
6. wrap 设置padding: 0, rightWidth, 0, leftWidth;
7. left 设置 margin-left: -100%, right 设置 margin-right: -rightWidth;
8. 给 left 设置position: relative; left: -leftWidth;

二、双飞翼布局

双飞翼布局和圣杯布局很类似，不过是在 .main 的div里又插入一个div，通过调整内部div的margin值，实现中间栏自适应，内容写到内部div中。

  • 实现步骤：
  1. 首先把left、main、right都放出来, main 中增加 content
  2. 给它们三个设置上float: left, 脱离文档流；
  3. 一定记得给 wrap 设置上 overflow: hidden; 可以形成BFC撑开文档
  4. left、right 设置上各自的宽度
  5. main 设置width: 100%;

接下来与圣杯布局不一样的地方：
6. left设置 margin-left: -100%, right设置 margin-right: -rightWidth;
7. content 设置 padding: 0, rightWidth, 0, leftWidth;

双飞翼布局省去了很多css，而且由于不用使用定位，可以获得比圣杯布局更小的最小宽度；双飞翼布局会一直随着浏览器可视区域宽度减小从而不断挤压中间部分宽度。所以需要设置给页面一个min-width > LeftWidth + RightWidth；

关于负 margin

margin-top 为负值 设置该属性的元素向上移动
margin-left 为负值 设置该属性的元素向左移动
margin-bottom 为负值 设置该属性的元素不动，下边的元素向上移动
margin-right 为负值 设置该属性的元素不动，右边的元素向上移动（或者看作该元素的宽度变小）
