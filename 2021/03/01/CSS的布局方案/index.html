CSS的布局方案

发表于2021-03-01|更新于2021-03-03|CSS
|字数总计:1.5k|阅读时长:7分钟|阅读量:

一、水平居中/垂直居中

1. 水平居中

行内元素水平居中

这里行内元素指的是文本（text）、图像（img）、按钮超链接、input等，只需给父元素设置text-align: center; 即可实现

块级元素水平居中

  • 块级元素有明确的宽度
  1. 只需给需要居中的块级元素设置 margin: 0 auto; 即可
  2. 通过绝对定位和 margin-left: -width/2进行水平居中, 前提是父元素position: relative（子绝父相）
  • 块级元素宽度未知
  1. 通过绝对定位 + transform 的方式
  2. flex 布局(给父元素设置)
  3. table布局
  4. inline-block(多个块状元素，父子关系)实现水平居中

2. 垂直居中

  1. 纯文本设置垂直居中

    通过设置line-height = height 或 padding-top = padding- bottom 实现垂直居中。（这两个一个原理)

  2. 绝对定位 + transform

  3. 绝对定位 + margin

  4. 通过设置vertical-align: middle;

    vertical-align: middle使用的前提条件是内联元素以及display值为table-cell的元素

  5. flex弹性布局

    在需要垂直居中的父元素上设置：display：flex；和 align-items：center；

3. 水平垂直居中

  1. 基于定位实现的
  • 绝对定位 + margin: auto
  • 绝对定位 + 负 margin
  • 绝对定位 + transform
  1. flex布局
  2. table布局

4.通过浮动实现

通过浮动实现的三栏布局有一个很明显的缺点，就是我们的代码层面上来讲 html 内容结构不正确，我们必须把 .center 元素放在最下面而不是中间位置，这是 float 所产生的的布局影响所导致的。

5.通过定位实现

思路
1.父元素相对定位，通过绝对定位将左右两栏固定
2.通过 margin 设置左右边距，留出内容块
